# SaaS 电商平台（多端前台 + 用户管理后台）

**项目描述**

面向多商户的 SaaS 电商平台，包含消费者使用的商城前台及商户运营后台。前台基于 Vue3 + Vite + uni-app 构建，实现 Web 与微信小程序一套代码多端运行；后台基于 Nuxt3 搭建，提供商品管理、订单管理、用户管理及运营配置等能力，满足多租户商户的差异化业务需求。

**技术栈**

Vue 3 / Vite / TypeScript / uni-app / Pinia / Nuxt 3

**主要职责与亮点**

- 基于 Vue3 + Vite + uni-app 搭建商城前台基础架构，实现 Web 与微信小程序多端统一开发
- 负责商品列表、商品详情、购物车、订单结算、支付流程、用户中心等核心业务模块开发
- 设计多端兼容方案，统一页面组件、状态管理、接口请求层及公共业务逻辑，实现 一套代码多端运行
- 封装平台能力适配层，兼容 Web 与小程序在登录、支付、缓存、文件上传等场景下的差异化 API
- 基于 Pinia 搭建全局状态管理，统一购物车、用户信息及订单状态流转逻辑
- 基于 Nuxt3 开发商户运营后台，负责商品管理、订单管理、用户管理及营销配置模块
- 推动前端工程化建设，包括组件库沉淀、请求封装、路由权限控制及性能优化方案
- 通过代码复用与组件抽象，提升多端业务代码复用率，降低项目迭代与维护成本

**项目成果**
支持 50+ 商户接入，前端多端代码复用率提升至 70%，显著降低项目交付成本
