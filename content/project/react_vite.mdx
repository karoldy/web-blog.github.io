# Vite + react

node版本 > 18

## 创建一个vite项目

按照提示选择 React 和 TypeScript(是否需要)

```bash
npm create vite@latest
```

## 安装 Tailwind CSS

按照 tailwind 官网提示安装 `tailwindcss`, `@tailwindcss/vite`

```bash
npm install tailwindcss @tailwindcss/vite
```

在`vite.config.ts`文件加入 `tailwindcss`

```ts
import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite'
export default defineConfig({
  plugins: [
    tailwindcss(),
  ],
})
```

在 `index.css` 或 `App.css` 加入 `@import "tailwindcss";`

```css
@import "tailwindcss"
```

## 配置路由

安装 react-router-dom

```bash
npm install react-router-dom
```

```bash
npm install @types/react-router-dom --save-dev
```

`src/router.ts`

```ts
import { createBrowserRouter, RouteObject } from 'react-router-dom;
import { lazy, Suspense } from 'react;
import Layout from '@/components/Layout;

// 定义一个 LazyLoad 组件，封装 Suspense Loing组件可以自定义，这里先用文本代替
const LazyLoad = (Component: React.LazyExoticComponent<React.FC>) => (
  <Suspense fallback={<div className="text-center p-4">加载中...</div>}>
    <Component />
  </Suspense>
);

const Home = lazy(() => import('@/pages/Home'));
const NotFound = lazy(() => import('@/pages/NotFound'));

// 定义路由配置
const routes: RouteObject[] = [
  {
    path: "/",
    element: <Layout />,
    children: [
      { index: true, element: LazyLoad(Home) }, // 首页 "/"
      { path: "*", element: LazyLoad(NotFound) }, // 404 页面
    ],
  },
];

// 创建路由
const router = createBrowserRouter(routes);

export default router;
```

在 `App.tsx` 使用 `RouterProvider`

```ts
import { RouterProvider } from 'react-router-dom';
import router from './router';

const App = () => {
  return <RouterProvider router={router} />;
};

export default App;
```

## 配置环境变量

安装 env-cmd

```bash
npm install env-cmd --save-dev
```

在根目录下创建 `.env.dev`, `.env.uat`, `.env.prod`文件

`.env.dev`

```
VITE_API_URL=http://example-dev.com
```

`.env.uat`

```
VITE_API_URL=http://example-uat.com
```

`.env.prod`

```
VITE_API_URL=http://example.com
```

### 配置 env-cmd 在 package.json 中使用

```json
{
  "scripts": {
    "dev": "env-cmd -f .env.dev vite",  // 开发环境
    "build": "env-cmd -f .env.prod vite build",  // 生产环境构建
    "serve": "vite preview"  // 预览
  }
}
```

### 在项目中使用变量

在 Vite 中，环境变量必须以 `VITE_` 开头才能在客户端访问。

例如，在你的代码中，你可以使用 `import.meta.env` 来访问这些变量：

```tsx
import React from 'react';

const App = () => {
  return (
    <div>
      <p>API URL: {import.meta.env.VITE_API_URL}</p>
    </div>
  );
};

export default App;
```

## 路径别名

`vite.config.ts`

```ts
resolve: {
  alias: {
    '@': path.resolve(__dirname, './src'),
  },
},
```

## 安装Sass

vite内置了sass解析和css模块化，所以安装一个sass包即可

```bash
npm install sass --save-dev
```

在 `src` 新建一个styles文件夹放sass变量和全局样式

`src/styles/_base.scss`

```scss
.test {
  color: $primary-color;
}
```

`src/styles/_variables.scss`

```scss
$primary-color: blue;
```

`src/styles/_index.scss`

```scss
@forward "base";
@forward "variables";
```

在需要的组件内使用即可

```scss
@use '@/styles/index';
```
