# vite + vue2

## 目录结构

```
vue2-vite-app
├── node_modules
├── src
│   ├── App.vue
│   └── main.js
└── public
|   └── favicon.ico
├── index.html
├── postcss.config.js
├── vite.config.js
├── package.json
```
